# freshbiteMe
FreshBite est un site web fictif simple pour un restaurant proposant des repas frais et sains. Le projet utilise HTML, CSS et JavaScript basique pour créer une interface responsive avec navigation, menu, galerie et formulaire de contact.
# Pages Incluses
- index.html: Page d'accueil avec section hero.
- menu.html: Liste des plats sous forme de cartes.
- gallery.html: Grille d'images avec légendes.
- contact.html: Formulaire de contact avec validation.
# Comment Visualiser Localement
1. Ouvrir le dossier `freshbite/` dans votre navigateur.
2. Ouvrir `index.html` pour commencer.
3. S assurer que les images sont placées dans `assets/img/` (utilisez des placeholders ou vos propres images).
# Fonctionnalités JS
- Menu burger responsive : Ouvre/ferme le menu sur mobile, avec fermeture au clic sur lien ou en dehors.
- Validation du formulaire : Vérifie les champs nom, email et message, affiche des messages d'erreur.
# Limitations Connues
- Pas d'envoi réel du formulaire (simulation via alert).
- Hébergement : public.
